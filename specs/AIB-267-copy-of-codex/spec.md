# AIB-267: Legal pages - Terms of Service & Privacy Policy

## Description
Add mandatory legal pages for a public SaaS application.

## Pages
- **Terms of Service**: usage conditions, liability limitation, BYOK (user responsible for API costs), AI-generated data
- **Privacy Policy**: collected data (GitHub profile, email), cookies (NextAuth session), no data resale, GDPR (right to deletion)

## Behavior
- Pages accessible without login (public routes)
- Links in footer on all pages
- Links displayed before first sign-up (consent)
- Content in English
- AI-generated code is user's responsibility
