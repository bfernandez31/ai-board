## Technical Notes

**Why AIB-142 is sufficient**:
The bug was autocomplete persisting after command selection. Root cause: no check for spaces in query string. AIB-142's solution:
```typescript
if (!query.includes(' ')) {
  // Show autocomplete
}
```

This prevents autocomplete from appearing when the query contains spaces (i.e., after selection adds space). Simple, correct, tested.

**AIB-143 scope creep**:
Added viewport positioning to prevent overflow at modal edges. While useful, this wasn't part of the original bug report and lacks test coverage. Feature mixing makes rollback harder if issues arise.
