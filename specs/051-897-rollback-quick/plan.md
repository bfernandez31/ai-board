## Summary

Enable rollback of tickets from BUILD stage to INBOX stage when quick-impl or normal workflow jobs fail or are cancelled. This provides users a recovery mechanism to retry with different workflow paths after failures, ensuring clean state reset (workflowType → FULL, branch → null, version → 1, failed job deleted) for successful restart without state conflicts.
