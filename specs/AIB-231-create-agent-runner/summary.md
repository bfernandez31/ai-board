**Branch**: `AIB-231-create-agent-runner` | **Date**: 2026-03-06

## Changes Summary

Created unified `.github/scripts/run-agent.sh` shell script that abstracts CLI installation, authentication, telemetry configuration, and command invocation across Claude Code and Codex CLI agents. Updated all 6 GitHub workflow files (speckit, quick-impl, verify, cleanup, ai-board-assist, iterate) to use this script instead of hardcoded Claude-specific steps. Added `agent` input to quick-impl.yml.

## Key Decisions

- Implemented all script functions (validate_auth, install, invoke, telemetry, AGENTS.md generation) in a single file for simplicity. Codex telemetry mapped to `~/.codex/config.toml`. AGENTS.md generated by copying CLAUDE.md with 32KB truncation. CLI binary existence verified after `bun add -g` install. All workflows call `../ai-board/.github/scripts/run-agent.sh` with agent type from inputs.
